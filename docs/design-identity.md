# Design Identity — Agent Ctx

## Personality
- Mood: techy
- Density: spacious
- Shape: sharp (2px border-radius)
- Tone: dark
- Design direction: "Dark, clean developer tool with monospace terminal elements. Emphasizes clarity and context — the design itself should feel like organized information, not chaos."

## Color Palette
- primary:     #6C63FF  — purple accent (AI/agent feel)
- secondary:   #00D9A3  — green (success, sync, connection)
- accent:      #FF6B6B  — coral red (highlights, badges, decisions)
- background:  #0D1117  — GitHub-dark background
- surface:     #161B22  — card/panel backgrounds
- text:        #E6EDF3  — primary text
- text-muted:  #7D8590  — secondary text
- terminal-bg: #0D1117  — terminal preview background
- terminal-fg: #E6EDF3  — terminal preview text

## Typography
- heading-font: Inter (Google Font)
- body-font:    Inter (Google Font)
- mono-font:    JetBrains Mono (code/terminal)

## Landing Page Variant
- Chosen: L2 — "Demo First"
- Reason: Agent Ctx is best understood by seeing the terminal workflow in action — snap → load → context appears. The hero IS the demo showing how context persists across sessions.

## Feature Presentation
- Alternating left-right sections (terminal output + text explanation)
- Each section shows one command with real terminal output on one side and a plain-English explanation on the other
